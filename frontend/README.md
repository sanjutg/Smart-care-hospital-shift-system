# Frontend - Hospital Shift Management System

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

Frontend runs on http://localhost:3000

Make sure the backend server is running on port 5000.

## Project Structure

- `src/components/Dashboard.jsx` - Main dashboard component
- `src/components/ConstantCarePatients.jsx` - Displays patients needing constant care
- `src/components/AvailableStaff.jsx` - Displays available doctors and nurses
